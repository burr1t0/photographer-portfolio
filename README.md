# Photographer Portfolio — Демо-сайт фотографа

Демонстрационный проект для портфолио. Минималистичный сайт-портфолио для фотографа.

## Живое демо

[Открыть сайт](https://burr1t0.github.io/photographer-portfolio)

## Что реализовано

- Hero-секция с именем и фоновым фото
- Масонри-галерея работ
- Блок "Обо мне"
- Пакеты услуг с ценами
- Контактная информация
- Адаптивная вёрстка (мобильные и десктоп)
- Плавные переходы и анимации
